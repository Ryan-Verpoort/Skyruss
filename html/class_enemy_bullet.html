EnemyBullet::EnemyBullet ( ObjectsPosition  enemyPos,
Screen  screen 
)

Default constructor for the EnemyBullet class.

Allows for the creation of all Enemy Bullets. On every game loop funtions are called in order to query and evaluate the current on-screen status of the bullet to be used during collision detection Sets the inital and current x and y positions of the bullet to the current and initial x and y positions of its rendered Enemy Sets the angle of the bullet to be the same as the Enemy at time of creation Sets the radius of the bullet for collision detection
