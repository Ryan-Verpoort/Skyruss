Detailed Description

ObjectPostion Class.

This class provides all information neccessary in determing and setting the location of all objects available during run-time. It is responsible for setting the initial postion of the created object, as well as object specific parameters such as object radius or angle

Additionally, it is responsible provding the information it sets to functions using the lacational information of the object during collision detection
