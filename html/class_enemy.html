Enemy::Enemy ( Screen  screen)

Default Constructor for the Enemy class.

Allows for the creation, destruction and rendering of Enemy objects On every Game loop, Enemy class functions are called to query the current position and status of on-screen rendered Enemy objects Used to set the initial x and y positions of each creted Enemy craft, as well as the current x and y positions of each Enemy craft to the centre of the screen Sets the angle at which the created Enemy craft is to move, as well as the craft's radius
